## Residual Risk

- The `file_path` rendering uses Python's str representation of the
  `tool_input.file_path` value, which on Windows includes backslashes.
  Operators on Windows will see `file_path=bridge\test-fake-pending-preflight-001.md`;
  on Unix, `bridge/test-fake-pending-preflight-001.md`. The regression
  test asserts the path substring is present after normalization (the
  test fixture uses forward slashes and the hook receives the same
  literal path); cross-platform diagnostic clarity is preserved.
- Active and template hooks are kept in lockstep via SHA256 parity. A
  future template-hook regeneration tool could codify this.
