Agent Red Logo Export Tool

Generate all required logo assets from the Beacon master design

How to use this tool:
  1. Open this file in a modern browser (Chrome recommended)
  2. Wait for the Inter font to load (previews will render correctly)
  3. Click "Download" on individual assets, or "Export All" to download everything
  4. Each file is named with its intended use (e.g., favicon-16x16.png)

To update the logo: Edit the drawIcon() and drawWordmark() functions in this file, then re-export all sizes.
