When presenting proposal items for prioritization, use this format for each item:

1. Brief description of what it does and why
2. Table with Desirability (very high / high / medium / low / very low) + Rationale
3. Table with Importance (very high / high / medium / low / very low) + Rationale
4. Summary table at the end with all items side by side

**Why:** Owner said "This is an excellent format. Please use this format in future reviews."

**How to apply:** Every proposal prioritization walk-through (Phases 0-4 and beyond) should use this format. Include code change impact column in summary.
