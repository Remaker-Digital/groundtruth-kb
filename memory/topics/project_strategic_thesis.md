Agent Red is the first project built on the real IP — the development pipeline (groundtruth-kb, governance framework, Loyal Opposition model, specification-driven workflow). Agent Red validates the pipeline; the pipeline is what produces commercially viable SaaS services at low cost and high velocity.

**Why:** The owner (Mike) made this explicit on 2026-04-10: "Agent Red is of no value without the pipeline that made it possible to build, operate and maintain Agent Red." The pipeline (tools, processes, metrics, prioritization framework, quality and oversight) is the valuable IP. Agent Red's commercial success is the first major test of whether the pipeline can reliably produce commercially viable SaaS.

**How to apply:** When presenting to investors/customers, frame Agent Red as proof-of-pipeline, not as the end product. The due diligence story is: "We built a pipeline that produces SaaS products with 0.97% post-ship defect rates and full audit trails. Agent Red is the first product. The pipeline scales to N products." All documentation, wiki pages, and presentations should lead with pipeline capability, not Agent Red features.
